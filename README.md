### 👋 About Me

Computer Science student at **TU Munich** and backend engineer with a focus on scalable, production-grade systems. I started out with Java tutorials on YouTube — now I architect distributed services, design infrastructure on Kubernetes, and build full-stack platforms. I care about writing clean code, understanding systems deeply, and not over-engineering before it's necessary.

- 🔭 Currently building backend infrastructure with **Go microservices** and **Kubernetes**
- 🏢 Working on Java/Spring Boot microservices at **Check24**
- 🌱 Exploring **service mesh** (Linkerd, mTLS), **CI/CD pipelines**, and **self-hosted auth**
- 💬 Ask me about **Java**, **Spring Boot**, **Go**, **Kubernetes**, **MySQL**, or **distributed systems**
- ⚡ Strong foundation in formal CS — discrete structures, algorithms, computer architecture
